# Gatsby Overview

Gatsby is a modern framework for building fast, secure, and powerful websites and applications. It combines the best parts of React, GraphQL, and modern web tooling into a single developer experience.

## Why Gatsby?
- **Performance by default**: Gatsby sites are optimized for speed out of the box.
- **Scalable**: From personal blogs to enterprise apps, Gatsby can handle it all.
- **Secure**: With pre-rendered pages and a decoupled architecture, security is easier to maintain.
- **Rich plugin ecosystem**: Thousands of plugins help you add functionality quickly.
- **Great developer experience**: Hot reloading, fast builds, and helpful error messages make development enjoyable.

## Getting Started
To create a new Gatsby site, run:
```bash
npm install -g gatsby-cli
gatsby new my-site
cd my-site
gatsby develop
